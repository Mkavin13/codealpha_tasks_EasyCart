# 🛍️ Easy Cart - Complete E-commerce Solution

A professional, full-stack e-commerce website built with HTML, CSS, JavaScript, Express.js, and MySQL.

![Easy Cart Preview](https://via.placeholder.com/800x400?text=Easy+Cart+Preview)

## ✨ Features

- **🛍️ Product Listings** - Beautiful grid layout with product images, descriptions, and prices
- **🔍 Product Search** - Search and filter products by category, price range
- **🛒 Shopping Cart** - Add/remove items, update quantities, persistent cart
- **📱 Product Details** - Detailed product views with stock information
- **👤 User Authentication** - Secure registration and login with JWT
- **💳 Order Processing** - Complete checkout flow with order confirmation
- **📦 Order History** - View past orders and their status
- **🎨 Professional Design** - Clean, modern interface with custom color scheme
- **📱 Responsive** - Works perfectly on desktop, tablet, and mobile

## 🛠️ Tech Stack

### Frontend
- HTML5
- CSS3 (Custom properties, Flexbox, Grid)
- Vanilla JavaScript (ES6+)
- Responsive Design

### Backend
- Node.js
- Express.js
- MySQL (with mysql2)
- JWT Authentication
- bcrypt for password hashing
